About

VPHMAT is an open-source MATLAB package for phonon visualization and post-processing toolkit for QE, VASP, and Wannier90. The name ‘VPHMAT’ stands for Visualization of PHonon properties within the MATrix framework, targeting computational studies in condensed matter physics.

Features

  • User-friendly interface: Intuitive MATLAB-based environment

  • Computational efficiency: Optimized matrix operations for fast calculations

  • Publication-ready graphics: Generate high-quality figures suitable for academic publications

VPHMAT can be used to:

  • Visualize phonon modes

  • Plot band structure and DOS

  • Plot Fermi surface
